## Final Project To-Doist

Libraries: tkinter, datetime, calendar, threading
Google API Imports w/ os, pickle

Resources:
https://docs.python.org/3/library/tkinter.html
https://www.youtube.com/watch?v=dqiHmglIQSE&t=678s
Claude -> for set up/ questions
Google -> how to use and apply API

How to use (set up):
1. Install dependencies: pip install google-auth google-auth-oauthlib google-api-python-client
2. Get a credentials.json from Google Cloud Console (enable Calendar API -> create OAuth credentials)
3. Place credentials.json in the same folder as the script
4. Run: python3 To-doist.py: a browser window will pop up to sign into Google and you all set!
